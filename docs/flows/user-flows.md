# User Flows

## Core user journeys

### Authentication

- register account
- sign in
- restore session on app relaunch
- recover from token expiry

### Messaging

- open conversation list
- start one-to-one conversation
- send text message
- send file attachment
- receive reply
- handle offline sync

### Device management

- inspect active devices
- revoke a device session
- mark a device trusted

### Future remote control

- request remote session from chat
- approve or reject incoming request
- monitor active session
- end session explicitly
